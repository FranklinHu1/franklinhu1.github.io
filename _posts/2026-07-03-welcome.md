This is the first post on my new site. Jekyll turns any Markdown file placed in
the `_posts` directory (named `YYYY-MM-DD-title.md`) into a blog post.

You can write **Markdown**, embed code:

```python
def hello(name: str) -> str:
    return f"Hello, {name}!"
```

and include math, images, and links. Delete this file once you're ready to add
your own content.
